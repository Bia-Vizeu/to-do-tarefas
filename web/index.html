Início Usuários Tarefas

Bem-vindo ao Sistema de Tarefas

Esta aplicação foi desenvolvida para uma indústria do ramo alimentício que deseja aprimorar seu controle de tarefas entre os setores.

Sobre o Sistema

O sistema utiliza o modelo Kanban simplificado com três status principais: A fazer, Fazendo e Pronto. Ele permite:

  • Cadastro de usuários (nome, e-mail);
  • Cadastro de tarefas com setor, prioridade e status;
  • Visualização das tarefas separadas por status;
  • Atualização do status e edição/exclusão das tarefas.

Funcionalidades Disponíveis

  1. Cadastro de Usuários: Registre os colaboradores.
  2. Cadastro de Tarefas: Crie tarefas vinculadas aos usuários.
  3. Gerenciamento de Tarefas: Visualize, edite, exclua ou altere o status das tarefas.

Como usar?

Use o menu acima para navegar entre as seções. Comece cadastrando os usuários e depois crie as tarefas. Em seguida, acesse a tela de gerenciamento para acompanhar o andamento das atividades.
